# skills/

所有 skill 的正本 (source of truth)。每個子資料夾 = 一個 skill,內含必要的 `SKILL.md`。

- 新增:複製 [`../template/`](../template/) 到 `./<skill-name>/`,或用 `/skill-creator`。
- 測試 / 安裝:見根目錄 [`../README.md`](../README.md)(`install` copy 到各工具 `~/.<tool>/skills/` 或目標專案 skills 目錄)。
- 撰寫規範:見 [`../references/skill-authoring.md`](../references/skill-authoring.md)。
