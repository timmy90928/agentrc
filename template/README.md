# template/ — 新 skill 起手範本

[`SKILL.md`](SKILL.md) 是新建 skill 的空白範本。

- **新增 skill**:複製本資料夾到 [`../skills/`](../skills/)`<skill-name>/` 後改寫,或用 `/skill-creator`。
- **撰寫規範**:動手前先讀 [`../references/skill-authoring.md`](../references/skill-authoring.md)(權威版)。

> 提醒:`name` 僅小寫字母 / 數字 / 連字號、不可含 `anthropic` / `claude`;`description` 第三人稱、含「做什麼 + 何時用」;資料夾名 = frontmatter `name`。
